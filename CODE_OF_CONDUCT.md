# Code of Conduct

This project follows the Contributor Covenant, version 2.1.

The full text of the Code of Conduct is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project, you agree to abide by those terms.

## Reporting

Incidents that should be addressed under this code can be reported via
GitHub's abuse reporting mechanism or by opening a private security
advisory on the repository. Reports are handled confidentially.
